Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Definitions of allocator which is used to acquire memory resources in partition compilation and execution.

SYCL allocator (#make_allocator) should be used for SYCL runtime and host allocator should be used for non-SYCL.
